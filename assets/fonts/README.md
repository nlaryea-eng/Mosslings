# Mosslings Fonts

`moss-pixel.woff2` is a vendored copy of the Latin subset of Press Start 2P
from Google Fonts. It is loaded locally through `style.css`; the game has no
runtime remote font dependency.

To replace the display face, drop in a new `.woff2` with the same filename or
update the `@font-face` URL and cache-busting version in `style.css`.
